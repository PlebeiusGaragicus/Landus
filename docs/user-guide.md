# Landus — Player Guide

Landus is a two-player arcade cabinet. Pick a game from the carousel, play, hold ESC to come back.

## Controls

| Function | Player 1 | Player 2 |
|----------|----------|----------|
| Joystick | W / A / S / D | Arrow keys |
| Button 1 | F | K |
| Button 2 | G | L |
| Button 3 | T | O |
| Quit game | Hold ESC (1 sec) | Hold ESC (1 sec) |

(On the cabinet, the joysticks and buttons are wired to these keys — just use the sticks and buttons.)

## Picking a Game

1. Move the Player 1 joystick **left/right** to browse the carousel. Each game plays its theme song when highlighted.
2. Press any **Player 1 button** to launch the highlighted game.

## During a Game

- Each game shows its own instructions. Player count varies per game (check the card).
- **Hold ESC for 1 second** to quit back to the game selection screen at any time.

## If Something Looks Wrong

- **Game frozen?** Hold ESC for **3 seconds** — the system force-closes the game and returns to the menu.
- **"<game> crashed" banner** on the menu: the game exited with an error. Just pick it (or another game) again.
- Still stuck? Ask a staff member.
